# Fermentistry Webshop

## Deployment über GitHub Pages

1. Repository bei [github.com](https://github.com) erstellen
2. Diese ZIP entpacken & alle Dateien hochladen
3. Unter Settings > Pages > Source: Branch `main` + Ordner `root` auswählen
4. Dein Shop ist erreichbar unter `https://<username>.github.io/<repo>`

## Konfiguration:
- `bestellen.html`: Ersetze `YOUR_FORM_ID` durch dein Formspree-Formular
- PayPal: Ersetze `YOUR_PAYPAL_BUTTON_ID` durch Button-ID
